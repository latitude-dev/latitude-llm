# Latitude LLM Rewrite

This branch is the v2 rewrite workspace.

## Scope

- Legacy v1 code is not kept in this branch.
- New implementation lives directly in `apps/` and `packages/`.
- The rewrite focuses on human-aligned evals for agents.

## Commands

- `pnpm dev`
- `pnpm build`
- `pnpm lint`
- `pnpm format`
- `pnpm format:check`
- `pnpm typecheck`
- `pnpm test`
