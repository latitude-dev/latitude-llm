## New Features and Enhancements

1. **Evaluations V2 Refiner**:
   - Added Evaluations V2 to the Prompt Refiner, enhancing its functionality.
   - [PR #1104](https://github.com/latitude-dev/latitude-llm/pull/1104)

2. **Documents Queue/Worker**:
   - Introduced a queue and worker with higher concurrency for faster document processing.
   - [PR #1100](https://github.com/latitude-dev/latitude-llm/pull/1100)

3. **Experiments Schema**:
   - Implemented a new schema for experiments, affecting data handling.
   - [PR #1095](https://github.com/latitude-dev/latitude-llm/pull/1095)

## Improvements

4. **Stream Error Handling in SDK**:
   - Enhanced error handling in the SDK to address silent failures.
   - [PR #1090](https://github.com/latitude-dev/latitude-llm/pull/1090)

5. **Parameter Mapping to Datasets**:
   - Improved parameter mapping to datasets, along with UI enhancements.

## Minor Updates

- **Optional Heapdump Package**: Introduced an optional heapdump package for debugging.
- **Refactor Jobs**: Refactored jobs to remove meta programming, potentially addressing a memory leak.
