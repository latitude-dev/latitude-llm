## Usage

To trace Anthropic calls with Latitude, you need to install the Anthropic SDK
and initialize the Latitude SDK with your API key:

<CodeGroup>

```typescript Javascript
import * as Anthropic from '@anthropic-ai/sdk'
import { Latitude } from '@latitude-data/sdk'

new Latitude('your-api-key-here', {
  telemetry: {
    modules: {
      anthropic: Anthropic,
    },
  },
})
```

```python Python
from latitude_sdk import Latitude, LatitudeOptions
from latitude_telemetry import Instrumentors, TelemetryOptions

Latitude('your-api-key-here', LatitudeOptions(
  telemetry=TelemetryOptions(
    instrumentors=[
      Instrumentors.Anthropic,
    ],
  ),
))
```
