## Usage

To trace Azure OpenAI calls with Latitude, you need to install the Azure OpenAI SDK
and initialize the Latitude SDK with your API key:

<CodeGroup>

```typescript Javascript
import * as AzureOpenAI from '@azure/openai'
import { Latitude } from '@latitude-data/sdk'

new Latitude('your-api-key-here', {
  telemetry: {
    modules: {
      azureOpenAI: AzureOpenAI,
    },
  },
})
```

</CodeGroup>
